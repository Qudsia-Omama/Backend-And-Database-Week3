# Week 3: Back-End & Database - Authentication System

A full-stack authentication system with Node.js/Express backend and React frontend, featuring user registration, login, and MongoDB integration.

## Features

###  Backend Requirements
- **Node.js + Express App** - Complete server setup with middleware
- **User Signup/Login** - Full authentication flow with MongoDB
- **Password Encryption** - bcryptjs for secure password hashing
- **Basic Routing** - Organized Express routes with validation
- **Input Validation** - Comprehensive validation with express-validator

### Security Features
- Password hashing with salt rounds
- JWT token-based authentication
- Input sanitization and validation
- CORS enabled for frontend communication
- Environment variables for sensitive data

### Error Handling
- Validation errors with specific field messages
- Database error handling
- Network error responses
- Proper HTTP status codes


   ```bash

   ```
